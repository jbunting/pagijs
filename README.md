pagijs
======

A javascript library for pagi manipulation.

Building
--------

```
npm install
npm test
```


This library is primarily a port of jpagi. Some notes on translations follow:

`PagiSchemaLocator` is `function(uri) { ... }`. It returns a Schema object.
